# Miniprograms项目

## 概述:

这个是用来规避小程序审核的服务

基本使用方式为

1. 使用账号密码调用setStatus创建项目
2. 审核时修改项目状态为true，表示被影藏
3. 先checkStatus获取当前状态以隐藏无法通过审核的页面
4. 当审核通过后修改项目状态为false
5. 此时再访问就可以显示被隐藏的页面了
